# @dolores/core

Core types, hybrid retrieval (pgvector cosine + Postgres full-text via Reciprocal Rank Fusion), local/OpenAI embedder abstraction, and async LLM extraction pipeline for the dolores agent memory system.

All shared interfaces (`Embedder`, `Memory`, `Fact`, `DAEMON_ROUTES`) live here and are imported by every other package in the monorepo.

→ See the [root README](https://github.com/yeneryigitcelik-debug/dolores#readme) for full documentation and quick start.
